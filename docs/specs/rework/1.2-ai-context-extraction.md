# Story 1.2: AI Context Extraction

**Epic:** 1 — Reibungslose Erfassung & Smarte Veredelung
**Priorität:** Hoch | **Abhängigkeiten:** Story 1.1

## Ziel

Apple Intelligence extrahiert automatisch Kategorie, Dauer, Priorität und Energy-Level aus dem Rohtext. Ergebnisse werden als Vorschläge gespeichert — nicht als endgültige Werte.

## Realistischer Scope (Apple Intelligence)

Das On-Device-Modell (~3B Parameter) ist gut bei Extraktion und Klassifikation. Aus "Mama anrufen wegen Geburtstagsfeier nächste Woche" kann es zuverlässig extrahieren: Kategorie → Social, Urgency → bald, Dauer → 15 Min. Dafür eignet sich besonders der Content-Tagging-Adapter und Guided Generation (`@Generable`).

Das Modell kann NICHT: Komplexe Schlussfolgerungen, Weltwissen anwenden, Verhaltensvorhersagen treffen.

## Akzeptanzkriterien

- [ ] Jeder Task mit `lifecycleStatus == .raw` wird automatisch an Enrichment übergeben
- [ ] Extraktion läuft asynchron im Hintergrund, blockiert keine UI
- [ ] Extrahierte Felder: `category`, `estimatedDuration`, `importance`, `urgency`, `energyLevel`
- [ ] Ergebnisse als Vorschläge gespeichert (`suggested*`-Felder), nicht in Hauptfelder überschrieben
- [ ] Fallback ohne Apple Intelligence: Task bleibt `.raw`, Refiner zeigt leere Vorschläge
- [ ] Siri/Shortcuts: `CreateTaskIntent` als App Intent registriert
- [ ] Timeout: Max 5 Sekunden, danach Partial Results akzeptieren

## Technische Vorgaben

### Model-Änderung: `LocalTask`
```swift
var suggestedCategory: TaskCategory?
var suggestedDuration: Int?        // Minuten
var suggestedImportance: Int?      // 1-3
var suggestedUrgency: Int?         // 1-3
var suggestedEnergyLevel: Int?     // 1-3
```

### Guided Generation Struct
```swift
@Generable
struct TaskEnrichmentResult {
    @Guide(description: "Kategorie: income, maintenance, recharge, learning, giving_back")
    var category: String
    @Guide(description: "Geschätzte Dauer in Minuten, zwischen 5 und 240")
    var estimatedMinutes: Int
    @Guide(description: "Wichtigkeit 1-3, wobei 3 am wichtigsten")
    var importance: Int
    @Guide(description: "Dringlichkeit 1-3, wobei 3 am dringendsten")
    var urgency: Int
    @Guide(description: "Benötigtes Energielevel 1-3, wobei 3 am meisten Energie erfordert")
    var energyLevel: Int
}
```
