# RW 1.2 + 1.3 — AI Context Extraction + The Refiner (Implementation Spec)

## Approval

- [ ] Approved

## Purpose

Implements the complete "raw task refinement" pipeline: Apple Intelligence extracts structured metadata (category, duration, importance, urgency, energy level) from raw task titles and stores the results as `suggested*` fields. A new Refiner screen lets the user review, correct, and confirm each suggestion via swipe gestures, promoting tasks from `.raw` to `.active`.

This spec covers the concrete implementation of Stories 1.2 and 1.3 across all three delivery steps (Schema, Service, UI). It is derived from the feature specs at `docs/specs/rework/1.2-ai-context-extraction.md` and `docs/specs/rework/1.3-the-refiner.md`.

### Step 1 — Schema (~55 LoC, 2 files)

#### 1a. `LocalTask.swift` — Add `suggested*` properties

Add the following five properties to the `// MARK: - AI Task Scoring` section, immediately after the existing `needsTitleImprovement` field. All are `Optional` with no stored default so CloudKit migration receives `nil` for existing rows.

```swift
// MARK: - AI Suggestions (Refiner — RW 1.2)
// Written by SmartTaskEnrichmentService.enrichRawTask().
// Promoted to main fields on Refiner confirmation.
// All Optional + nil default required for CloudKit migration.

/// AI-suggested category rawValue (income/maintenance/recharge/learning/giving_back)
var suggestedCategory: String?

/// AI-suggested estimated duration in minutes (5–240)
var suggestedDuration: Int?

/// AI-suggested importance (1–3)
var suggestedImportance: Int?

/// AI-suggested urgency string ("urgent" / "not_urgent")
var suggestedUrgency: String?

/// AI-suggested energy level ("high" / "low")
var suggestedEnergyLevel: String?
```

Note: `suggestedCategory` uses `String?` (not `TaskCategory?`) to stay consistent with the existing `taskType: String` pattern and to avoid breaking CloudKit enum constraints.

Also add the `confirmSuggestions()` helper on `LocalTask`:

```swift
// MARK: - Refiner Confirmation

/// Promotes suggested* values to main fields and transitions status to .active.
/// Only overwrites main fields that are currently nil/empty (respects user edits).
/// No-op if lifecycleStatus is not "raw".
func confirmSuggestions() {
    guard lifecycleStatus == TaskLifecycleStatus.raw.rawValue else { return }

    if taskType.isEmpty, let cat = suggestedCategory {
        taskType = cat
    }
    if estimatedDuration == nil, let dur = suggestedDuration {
        estimatedDuration = dur
    }
    if importance == nil, let imp = suggestedImportance {
        importance = max(1, min(3, imp))
    }
    if urgency == nil, let urg = suggestedUrgency {
        urgency = urg
    }
    if aiEnergyLevel == nil, let energy = suggestedEnergyLevel {
        aiEnergyLevel = energy
    }

    lifecycleStatus = TaskLifecycleStatus.active.rawValue
    modifiedAt = Date()
}
```

**Add `enrichRawTask()` function** — writes to `suggested*` fields only, never overwrites main fields:

```swift
/// Enrich a raw task: writes AI output to suggested* fields.
/// Does NOT write to main fields (importance, urgency, taskType, etc.).
/// Call only for tasks with lifecycleStatus == .raw.
func enrichRawTask(_ task: LocalTask) async {
    guard Self.isAvailable else { return }
    guard AppSettings.shared.aiScoringEnabled else { return }
    guard task.lifecycleStatus == TaskLifecycleStatus.raw.rawValue else { return }

    #if canImport(FoundationModels)
    if #available(iOS 26.0, macOS 26.0, *) {
        await performRawEnrichment(task)
    }
    #endif
}

#if canImport(FoundationModels)
@available(iOS 26.0, macOS 26.0, *)
private func performRawEnrichment(_ task: LocalTask) async {
    let prompt = buildPrompt(for: task)

    do {
        let session = LanguageModelSession {
            "Du analysierst Task-Titel und leitest fehlende Attribute ab."
            ""
            "Wichtigkeit (1-3): 1=nice to have, 2=should do, 3=must do"
            "Dringlichkeit: Zeitkritische Begriffe (Termin, Frist, morgen, heute) = true"
            "Kategorie: income (Geld verdienen), maintenance (Pflege/Haushalt), recharge (Erholung), learning (Lernen), giving_back (Helfen)"
            "Energie: high = tiefe Fokus-Arbeit (Programmieren, Schreiben, Analyse), low = Routine (Einkaufen, Putzen)"
            "Dauer in Minuten: Realistisch fuer den Task, zwischen 5 und 240"
            ""
            "Liefere Vorschlaege — der Nutzer prueft sie im Refiner."
        }

        let response = try await session.respond(to: prompt, generating: TaskEnrichment.self)
        let result = response.content

        // Write to suggested* fields only — never overwrite main fields here
        task.suggestedImportance  = max(1, min(3, result.suggestedImportance))
        task.suggestedUrgency     = result.suggestedUrgent ? "urgent" : "not_urgent"

        let validTypes = ["income", "maintenance", "recharge", "learning", "giving_back"]
        task.suggestedCategory    = validTypes.contains(result.suggestedTaskType) ? result.suggestedTaskType : nil

        let energy = result.suggestedEnergyLevel.lowercased()
        task.suggestedEnergyLevel = (energy == "high" || energy == "low") ? energy : "low"

        let dur = max(5, min(240, result.suggestedDuration))
        task.suggestedDuration    = dur

        try modelContext.save()

    } catch let error as LanguageModelError where error == .guardrailViolation {
        // Guardrail false-positive: leave suggested* nil, task stays in Refiner
        print("[SmartEnrichment] Guardrail violation for task '\(task.title)' — skipping suggestions")
    } catch {
        print("[SmartEnrichment] enrichRawTask failed for '\(task.title)': \(error)")
    }
}
#endif
```

**Guard `.raw` tasks in `performBatchEnrichment()`** — add a filter condition so raw tasks are excluded from the existing enrichment path (they use `enrichRawTask` instead):

```swift
// Existing filter in performBatchEnrichment():
```

**Trigger on task creation** — wherever a new task is saved with `lifecycleStatus == .raw` (Quick Dump, Share Extension, Siri), call `Task { await enrichmentService.enrichRawTask(newTask) }` after the `modelContext.save()`.

#### 3a. `RefinerView.swift` (CREATE)

```swift
import SwiftUI
import SwiftData

struct RefinerView: View {
    @Query(
        filter: #Predicate<LocalTask> { $0.lifecycleStatus == "raw" && !$0.isCompleted },
        sort: \LocalTask.createdAt,
        order: .reverse
    ) private var rawTasks: [LocalTask]

    @Environment(\.modelContext) private var modelContext

    var body: some View {
        NavigationStack {
            Group {
                if rawTasks.isEmpty {
                    emptyState
                } else {
                    taskList
                }
            }
            .navigationTitle("Refiner")
            .toolbar {
                if !rawTasks.isEmpty {
                    ToolbarItem(placement: .primaryAction) {
                        Button("Alle bestätigen") {
                            confirmAll()
                        }
                        .accessibilityIdentifier("refiner_confirmAllButton")
                    }
                }
            }
        }
    }

    private var taskList: some View {
        ScrollView {
            LazyVStack(spacing: 12) {
                ForEach(rawTasks) { task in
                    RefinerTaskCard(task: task)
                        .padding(.horizontal)
                }
            }
            .padding(.vertical)
        }
        .accessibilityIdentifier("refiner_taskList")
    }

    private var emptyState: some View {
        VStack(spacing: 16) {
            Image(systemName: "sparkles")
                .font(.system(size: 48))
                .foregroundStyle(.secondary)
            Text("Alles veredelt")
                .font(.title2.weight(.semibold))
            Text("Neue Tasks aus dem Quick Dump erscheinen hier zur Prüfung.")
                .font(.subheadline)
                .foregroundStyle(.secondary)
                .multilineTextAlignment(.center)
        }
        .padding()
        .accessibilityIdentifier("refiner_emptyState")
    }

    private func confirmAll() {
        for task in rawTasks {
            task.confirmSuggestions()
        }
        try? modelContext.save()
    }
}
```

#### 3b. `RefinerTaskCard.swift` (CREATE)

```swift
import SwiftUI
import SwiftData

struct RefinerTaskCard: View {
    let task: LocalTask
    @Environment(\.modelContext) private var modelContext
    @State private var showDeleteConfirm = false

    var body: some View {
        VStack(alignment: .leading, spacing: 10) {
            // Raw title
            Text(task.title)
                .font(.body.weight(.medium))
                .accessibilityIdentifier("refinerCard_title")

            // AI suggestion chips
            if hasAnySuggestion {
                suggestionChips
            } else {
                Text("Keine KI-Vorschläge verfügbar")
                    .font(.caption)
                    .foregroundStyle(.secondary)
                    .accessibilityIdentifier("refinerCard_noSuggestions")
            }
        }
        .padding()
        .background(.regularMaterial, in: RoundedRectangle(cornerRadius: 14))
        .swipeActions(edge: .trailing, allowsFullSwipe: true) {
            Button(role: .destructive) {
                deleteTask()
            } label: {
                Label("Löschen", systemImage: "trash")
            }
            .accessibilityIdentifier("refinerCard_swipeDelete")
        }
        .swipeActions(edge: .leading, allowsFullSwipe: true) {
            Button {
                task.confirmSuggestions()
                try? modelContext.save()
            } label: {
                Label("Bestätigen", systemImage: "checkmark")
            }
            .tint(.green)
            .accessibilityIdentifier("refinerCard_swipeConfirm")
        }
        // VoiceOver button fallback
        .contextMenu {
            Button("Bestätigen") {
                task.confirmSuggestions()
                try? modelContext.save()
            }
            Button("Löschen", role: .destructive) {
                deleteTask()
            }
        }
    }

    private var hasAnySuggestion: Bool {
        task.suggestedCategory != nil ||
        task.suggestedDuration != nil ||
        task.suggestedImportance != nil ||
        task.suggestedUrgency != nil ||
        task.suggestedEnergyLevel != nil
    }

    private var suggestionChips: some View {
        FlowLayout(spacing: 6) {
            if let cat = task.suggestedCategory {
                SuggestionChip(
                    label: categoryLabel(cat),
                    icon: "tag",
                    accessibilityID: "refinerCard_chip_category"
                )
            }
            if let dur = task.suggestedDuration {
                SuggestionChip(
                    label: "\(dur) Min",
                    icon: "clock",
                    accessibilityID: "refinerCard_chip_duration"
                )
            }
            if let imp = task.suggestedImportance {
                SuggestionChip(
                    label: importanceLabel(imp),
                    icon: "flag",
                    accessibilityID: "refinerCard_chip_importance"
                )
            }
            if let urg = task.suggestedUrgency {
                SuggestionChip(
                    label: urg == "urgent" ? "Dringend" : "Nicht dringend",
                    icon: "bolt",
                    accessibilityID: "refinerCard_chip_urgency"
                )
            }
            if let energy = task.suggestedEnergyLevel {
                SuggestionChip(
                    label: energy == "high" ? "Hohe Energie" : "Niedrige Energie",
                    icon: "brain",
                    accessibilityID: "refinerCard_chip_energy"
                )
            }
        }
    }

    private func deleteTask() {
        modelContext.delete(task)
        try? modelContext.save()
    }

    private func categoryLabel(_ raw: String) -> String {
        switch raw {
        case "income":      return "Einnahmen"
        case "maintenance": return "Pflege"
        case "recharge":    return "Erholung"
        case "learning":    return "Lernen"
        case "giving_back": return "Helfen"
        default:            return raw
        }
    }

    private func importanceLabel(_ value: Int) -> String {
        switch value {
        case 1: return "Niedrig"
        case 2: return "Mittel"
        case 3: return "Hoch"
        default: return "\(value)"
        }
    }
}

/// Single suggestion chip with label and icon.
/// Tapping opens an inline picker (Phase 2 enhancement — not in RW 1.3 scope).
struct SuggestionChip: View {
    let label: String
    let icon: String
    let accessibilityID: String

    var body: some View {
        Label(label, systemImage: icon)
            .font(.caption.weight(.medium))
            .padding(.horizontal, 10)
            .padding(.vertical, 5)
            .background(.blue.opacity(0.12), in: Capsule())
            .foregroundStyle(.blue)
            .accessibilityIdentifier(accessibilityID)
    }
}
```

## Expected Behavior

- **Input:** A `LocalTask` with `lifecycleStatus == "raw"` (created via Quick Dump)
- **AI enrichment output:** `suggested*` fields populated on `LocalTask` within 5 seconds, or left `nil` if Apple Intelligence is unavailable or returns a guardrail violation
- **Refiner display:** All `.raw` tasks appear in `RefinerView` sorted newest-first; each card shows the raw title and AI suggestion chips
- **Confirm (swipe right / "Alle bestätigen"):** `confirmSuggestions()` is called — `suggested*` values are promoted to main fields, `lifecycleStatus` transitions to `"active"`, task disappears from Refiner and appears in Backlog
- **Delete (swipe left):** Task is permanently deleted from `ModelContext`
- **No AI available:** Tasks remain in Refiner with no chips; user can still confirm (main fields stay nil, status goes to `.active`) or delete
- **Side effects:**
  - `lifecycleStatus` change triggers `BacklogView` query re-evaluation (`.active` tasks become visible)
  - Tab badge count updates reactively via `@Query` on parent
  - `modifiedAt` is set to `Date()` on confirmation
  - CloudKit sync propagates all `suggested*` field changes

## Known Limitations

- **Inline chip editing (tap-to-change):** `SuggestionChip` tap-to-picker is deferred to a follow-up iteration. Chips are read-only in RW 1.3.
- **Foundation Models timeout:** There is no explicit 5-second timeout in `performRawEnrichment`. If the model hangs, the async task runs until completion. A `Task.sleep` timeout wrapper should be added in a follow-up.
- **`FlowLayout`:** If not already present in `Sources/Views/Components/`, a minimal wrapping layout must be created. This adds ~30 LoC not counted in the estimate.
- **macOS Refiner navigation:** Wiring the macOS Sidebar "Refiner" row to `NavigationSplitView`'s detail column is the responsibility of the Mac root view (not specified here). The SidebarView only emits the tap.
- **Badge on iOS Tab:** `.badge(_:)` modifier on TabView items requires iOS 15+. This is satisfied by the iOS 26 deployment target.
- **`--inject-raw-tasks` launch argument:** The UI test harness requires the app to seed test data on this flag. Implementation of the test data injection is a prerequisite for the UI tests to go GREEN.
- **`urgency` type mismatch:** `LocalTask.urgency` is `String?` ("urgent"/"not_urgent"), while `PlanItem.urgency` is also `String?`. The `suggestedUrgency` field follows the same convention. Callers must handle the string-to-display mapping themselves.
