# Story 1.3: The Refiner

**Epic:** 1 — Reibungslose Erfassung & Smarte Veredelung
**Priorität:** Hoch | **Abhängigkeiten:** Story 1.1, 1.2

## Ziel

Dedizierte Sektion zum Veredeln roher Tasks per Swipe-Geste: bestätigen, löschen oder korrigieren.

## Kontext

Neuer Screen. Zeigt Tasks mit `lifecycleStatus == .raw` und deren AI-Vorschläge.

## Akzeptanzkriterien

- [ ] Eigene Sektion in der Navigation (Badge zeigt Anzahl unveredelter Tasks)
- [ ] Zeigt alle Tasks mit `lifecycleStatus == .raw`, neueste zuerst
- [ ] Pro Task: Rohtext + AI-Vorschläge als editierbare Chips/Badges
- [ ] Swipe rechts = Bestätigen (Vorschläge → Hauptfelder, Status → `.active`)
- [ ] Swipe links = Löschen
- [ ] Tap auf Vorschlag öffnet Inline-Picker zum Korrigieren
- [ ] "Alle bestätigen"-Button für Batch-Verarbeitung
- [ ] Leerer State: Motivierende Nachricht
- [ ] Nach Bestätigung: Task erscheint im Backlog
