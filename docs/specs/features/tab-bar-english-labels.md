# Tab Bar English Labels

## Summary
Consistent English labels for the Floating Tab Bar with "Blox" as brand term.

## Changes

| Before | After |
|--------|-------|
| Backlog | Backlog |

## Rationale
- Consistent English UI
- Shorter labels (5-7 characters)
- "Blox" as unique brand term
